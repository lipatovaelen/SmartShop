# SmartShop
Интернет магазин смартфонов на платформе ASP.NET
## Функционал
* Регистрация
* Авторизация
* Панель администратора
* Возможность управлять ассортиментом
* Корзина
* Оформление заказа
* Отслеживание статуса заказа
* Поиск и сортировка
